# Painel de Acompanhamento — Hostel Anprado

Painel privado do Anderson para acompanhar o ciclo inicial de 16 a 30 de junho e as renovações mensais.

## Produção no Render

1. Criar um Web Service no Render a partir deste repositório.
2. Runtime: Node.
3. Build command: deixar vazio.
4. Start command: `node server.mjs`.
5. Adicionar a variável secreta `ANPRADO_ANDERSON_PASSWORD` no painel do Render.
6. Fazer o deploy e abrir a URL pública gerada.

O arquivo `render.yaml` contém a configuração equivalente para Blueprint.
